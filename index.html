RESPONSIVE DESIGN

All the sites that I develop are responsive for most device-widths. On requirement I also provide you with designs tailor made for specified device widths. Responsive sites are ranked better in SEO and go a long way in bettering your site's user experience.

UI UX USING FIGMA
